# FIX-ERROS — SE9 + SE7 (2026-06-19)

## Problemas Identificados

### 1. Title card: duração e transição
- **Atual:** 2-4 segundos, transição `fade` suave
- **Desejado:** 0.2s (6 frames), transição chamativa (`circleopen`)

### 2. Legendas no conteúdo
- **Atual:** `burn_subtitles` aplica SRT em TODO o vídeo
- **Desejado:** Sem legendas no conteúdo, só hook no title card

### 3. Erro 0x80004005 Windows
- **Causa raiz:** AAC sem `-profile:a aac_low` (Windows interpreta como HE-AAC)
- **Agravante:** 4 re-encodes do vídeo (segment→concat→subtitles→trim), sem `-profile:v main -level 4.0`, sem `-g 30`

### 4. SE7 lento (CPU)
- **Atual:** `DEVICE=cpu` no `.env`
- **Desejado:** GPU via `DEVICE=auto` + compose GPU

### 5. SE7 acentos PT-BR
- **Causa:** Modelo Chatterbox interno (código externo)
- **Mitigação:** Reduzir temperature (0.8→0.5), aumentar cfg_weight (0.5→0.7)
